# 🐍 Programming Fundamentals Lab (Python)

This repository contains structured lab tasks completed as part of learning programming fundamentals using Python.

The focus of these exercises is to build strong logic development skills and understand core programming concepts before moving into Object-Oriented Programming and Data Structures and Algorithms.

---

## 📚 Concepts Covered

### 🔹 Basic Programming
- Variables and Data Types
- Input / Output handling
- Operators and Expressions

### 🔹 Control Flow
- If / Else conditions
- Nested conditions
- Logical decision making

### 🔹 Loops
- For loops
- While loops
- Pattern-based problems
- Iteration-based logic building

### 🔹 Functions
- Function creation and usage
- Parameter passing
- Return values

### 🔹 Data Structures (Basics)
- Lists
- Tuples
- Dictionaries
- Basic operations on collections

### 🔹 Problem Solving
- Simple logic building exercises
- Number-based problems
- String manipulation basics
